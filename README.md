# Star Social

Star Social is social media software


Demo:
http://tinasnow.pythonanywhere.com/

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install Django and the following:
- django-bootstrap3
- django-braces
- misaka

## Usage

```bash
python manage.py runserver
```
